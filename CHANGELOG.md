# Changelog

All notable changes to the CollabEase (Excelerate MAD Flutter) project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2025-11 (Week 4 - Final Release)

### Added
- Complete navigation system using GoRouter
- Firebase Authentication integration (login and signup)
- Login screen with role selection (Learner/Educator)
- Signup screen with form validation
- Home dashboard with role-based UI
- Program listing screen with API integration
- Program details screen with comprehensive information
- Enrolled courses page with progress tracking
- Feedback form with rating and comment submission
- Data integration with JSONPlaceholder mock API
- Loading indicators and progress feedback
- Alert dialogs for user feedback
- Consistent branding and theme throughout the app
- Error handling and validation
- Screenshot documentation
- Comprehensive README with setup instructions

### Changed
- Updated main.dart to use MaterialApp.router
- Refactored navigation to use GoRouter
- Improved UI consistency across all screens
- Enhanced form validation

### Fixed
- Navigation flow between screens
- Data fetching and display issues
- Form submission handling
- Theme consistency
